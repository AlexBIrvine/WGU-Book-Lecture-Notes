# Module 20 - N-ary Relations

## 6.18 - N-ary equivalence relations

**Equivalence relation** has the following rules:

1. Reflexive
2. Transitive
3. Symmetric

It is represented by the notation $a~b$, which reads "a is equivalent to b"
