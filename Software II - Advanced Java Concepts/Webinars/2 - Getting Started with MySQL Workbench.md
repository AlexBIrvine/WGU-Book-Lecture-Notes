# How to use MySQL Workbench

### Connect:

Enter in credentials from the "book" lab.

Once connected, tell SQL what database to use:

```SQL
USE U05IAb;
```

### Display tables

To show table, then see datails:

```SQL
SHOW TABLES;
EXPLAIN <table name>;
```
