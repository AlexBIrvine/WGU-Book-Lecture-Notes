2.3 - The division algorithm

Number Theory - Branch of mathematics concerned with integers.

Integer Division - Division with remainder.

Input: Integers n and d > 0.
Output: q = n div d, and r = n mod d.

Case 1: n ≥ 0.'

q := 0
r := n
While ( r ≥ d )
      q := q + 1
      r := r - d
End-While
